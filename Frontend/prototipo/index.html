Cómo usar:

  1. Configuración: Pega tu URL de API y API Key en la sección ⚙️ arriba
  2. Ingresos: Sube una foto de cualquier objeto (frutas, verduras, etc). Rekognition la identificará automáticamente.
  3. Salidas: Usa el ID del producto (ej: "apple", "vegetable") para retirar unidades, o haz clic en un producto del inventario. Si bajan del umbral, recibe una alerta por email.
  4. Inventario: La pestaña "Inventario Actual" consulta GET /inventario y muestra todos los productos en DynamoDB.
  5. Alertas: Si tienes SNS suscrito a tu email, recibe notificaciones cuando el stock de un producto baja demasiado.
